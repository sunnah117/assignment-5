Impotant FAQ

How do you select an element from the DOM?

You can use methods like `getElementById()`, `querySelector()`, or `getElementsByClassName()` to select DOM elements based on their ID, class, or other attributes.

What is the Document Object Model (DOM)?

The DOM is a programming interface that represents the structure of an HTML or XML document as a tree of objects. It allows developers to manipulate elements, attributes, and content dynamically.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows you to handle events at a parent element level, taking advantage of event bubbling. It's useful for managing events efficiently, especially when dealing with dynamic content or multiple child elements.

How do you manipulate an element's attributes and styles using the DOM?

You can use methods like `setAttribute()`, `getAttribute()`, or directly modify properties like `element.style` to change attributes and CSS styles of a DOM element.
